It all starts with a Box.

What?

Yep.

Programming is a set of instructions that we ask the computer to run.

In the computer's brain, you want to keep track of the building blocks of your program.

And to do that, we put each building block in a box. And later, we use it to do cool things.

Remember:

  • You can put anything in a box
  • And you can later take it out and use it
  • Or, you can take it out, change it and put it back.
  • Every box has a unique name.
  • Every `thing` we put in a box has a `type`
  • Don't forget the `;` - in JavaScript land, `;` is the equivalent of full top `.`

More Arithmetic Fun!

Increment / Decrement

Say you just grew one year older. How do we take your age and add 1 to it?


var yesterdayAge = 16;
var todayAge = yesterdayAge + 1;
console.log(todayAge);

In short form ...


// notice the `++`
var yesterdayAge = 16;
var todayAge = yesterdayAge ++;
console.log(todayAge);

Types!

In JavaScript land, `variable`s have `types`. To get the type of a variable, we can use this:


typeof(variableName);


// example:
var number = (12 * 2) / 4 * 5 - 2 + 1;
console.log(typeof(number));

Remember!

In programming, any written word or text is considered of the `String` type.


var name = "Orandy";
// name is of type String - because it is a word.

To mark a variable as string, we should wrap its value in `'` or `"` (single or double quotation mark).
